# KS Die Crafts

Marketing website for KS Die Crafts — precision die manufacturing for jewellery, coins, religious items, and custom applications.

## Stack

- [Next.js](https://nextjs.org/) 16
- React 19
- Tailwind CSS 4
- TypeScript

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command        | Description              |
| -------------- | ------------------------ |
| `npm run dev`  | Start development server |
| `npm run build`| Production build         |
| `npm run start`| Start production server  |
| `npm run lint` | Run ESLint               |

## Environment

Copy `.env.example` to `.env.local` and fill in any required values before running locally.
